1. Настройка проекта

  • Добавьте 'django.contrib.staticfiles' в список INSTALLED_APPS в вашем файле settings.py.
  • Установите переменную STATIC_URL, которая определяет URL, по которому статические файлы будут доступны. Обычно это '/static/'.

2. Организация файлов

Создайте папку static в корне вашего приложения или проекта. Внутри нее организуйте файлы по типу или функциональности (например, css/, js/, images/).

3. Подключение файлов в шаблонах

Используйте тег {% load static %} в начале ваших HTML-шаблонов. Вставляйте статические файлы с помощью тега {% static 'path/to/your/file' %}.

<link rel="stylesheet" type="text/css" href="{% static 'css/style.css' %}"><script src="{% static 'js/script.js' %}"></script>

4. Сборка статических файлов

  • Для разработки обычно достаточно использовать runserver, который автоматически обслуживает статические файлы.
  • Для продакшена используйте команду python manage.py collectstatic, которая собирает все статические файлы в одну директорию, определенную в STATIC_ROOT, для эффективного обслуживания.

5. Настройка для продакшена

Убедитесь, что веб-сервер (например, Nginx или Apache) настроен для обслуживания статических файлов из директории STATIC_ROOT.
